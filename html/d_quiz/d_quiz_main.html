홈 퀴즈 만들기 라이브러리 마이 퀴즈 퀴즈 리포트

HOW TO USE

D퀴즈는 선생님이 직접 만드시거나, 라이브러리를 통해 저장하신 퀴즈를 학생들의 직접 참여로 진행 할 수 있는 프로그램 입니다.
퀴즈 진행 완료 후 퀴즈 리포트를 통해 더욱 상세한 결과를 확인해 보세요!

퀴즈 만들기
  • 과목, 학년/학기, 제목, 문항 수 입력
  • 퀴즈 문제 및 답변 작성

퀴즈 유형 : O X / 선다형 / 단답형 / 초성형 / 매칭형

직접 만드신 퀴즈를 라이브러리 공유 하시면 다른 선생님들도 이용하실 수 있습니다.

퀴즈 플레이
  • 마이 퀴즈에서 진행할 퀴즈 ‘플레이'
  • QR코드 및 URL 복사를 이용하여
    학생에게 공유
  • 퀴즈 완료 후 결과 및 리포트 확인
